# Exercícios Personalizados para Alunos

Este documento contém exercícios personalizados para Guilherme de Lima Rocha e Suellen Araujo, baseados em suas necessidades de aprendizado.

---

## Exercícios para Guilherme de Lima Rocha

### 1. Validador de Senha
Crie um programa que valide uma senha com as seguintes regras:
- Deve ter no mínimo 8 caracteres.
- Deve conter pelo menos uma letra maiúscula.
- Deve conter pelo menos um número.

### 2. Calculadora de IMC
Escreva um programa que calcule o Índice de Massa Corporal (IMC) de uma pessoa e classifique-a de acordo com a tabela:
- Abaixo de 18.5: Abaixo do peso
- 18.5 a 24.9: Peso normal
- 25 a 29.9: Sobrepeso
- 30 ou mais: Obeso
- **Fórmula:** IMC = peso (kg) / altura² (m)

### 3. Conversor de Moeda
Crie um programa que converta um valor em Reais para Dólares e Euro, utilizando taxas de câmbio fixas.
- **Taxas de câmbio:** 1 USD = 5.0 BRL, 1 EUR = 6.0 BRL

### 4. Validador de Palíndromo
Escreva um programa que verifique se uma palavra ou frase é um palíndromo (lê-se da mesma forma de trás para frente, desconsiderando espaços).

### 5. Contador de Vogais
Crie um programa que conte quantas vogais (a, e, i, o, u) existem em uma string.

### 6. Calculadora de Fatorial
Escreva um programa que calcule o fatorial de um número.
- **Fórmula:** n! = n * (n-1) * (n-2) * ... * 1

### 7. Gerador de Tabuada
Crie um programa que gere a tabuada de um número de 1 a 10.

### 8. Calculadora de Média Ponderada
Escreva um programa que calcule a média ponderada de três notas, com pesos 2, 3 e 5 respectivamente.
- **Fórmula:** Média Ponderada = (nota1 * peso1 + nota2 * peso2 + nota3 * peso3) / (peso1 + peso2 + peso3)

### 9. Verificador de Número Primo
Crie um programa que verifique se um número é primo.
- **Dica:** Um número é primo se for divisível apenas por 1 e por ele mesmo.

### 10. Inversor de String
Escreva um programa que inverta uma string.

### 11. Calculadora de Juros Compostos
Crie um programa que calcule o valor final de um investimento com juros compostos.
- **Fórmula:** Valor Final = Valor Inicial * (1 + taxa de juros)^tempo

### 12. Contador de Palavras
Escreva um programa que conte quantas palavras existem em uma string.

### 13. Verificador de Ano Bissexto
Crie um programa que verifique se um ano é bissexto.
- **Regras:** Um ano é bissexto se for divisível por 4, mas não por 100, ou se for divisível por 400.

### 14. Calculadora de Desconto Progressivo
Escreva um programa que aplique um desconto progressivo com base no valor da compra:
- Até R$ 100: 5% de desconto
- De R$ 101 a R$ 500: 10% de desconto
- Acima de R$ 500: 15% de desconto

### 15. Simulador de Eleição
Crie um programa que simule uma eleição com três candidatos. O programa deve contar os votos e determinar o vencedor.

## Exercícios para Suellen Araujo

### 1. Calculadora de Troco
Escreva um programa que calcule o troco de uma compra, recebendo o valor da compra e o valor pago.

### 2. Verificador de Maioridade
Crie um programa que verifique se uma pessoa é maior de idade.

### 3. Conversor de Temperatura
Escreva um programa que converta uma temperatura de Celsius para Fahrenheit e vice-versa.
- **Fórmula:** F = C * 9/5 + 32, C = (F - 32) * 5/9

### 4. Calculadora de Média
Crie um programa que calcule a média de três notas.
- **Fórmula:** Média = (nota1 + nota2 + nota3) / 3

### 5. Verificador de Par ou Ímpar
Escreva um programa que verifique se um número é par ou ímpar.

### 6. Calculadora de Área de Círculo
Crie um programa que calcule a área de um círculo, recebendo o raio.
- **Fórmula:** Área = π * raio²

### 7. Contador de Dígitos
Escreva um programa que conte quantos dígitos existem em um número.

### 8. Verificador de Triângulo
Crie um programa que verifique se três lados formam um triângulo.
- **Regras:** A soma de dois lados deve ser maior que o terceiro lado.

### 9. Calculadora de Salário Líquido
Escreva um programa que calcule o salário líquido de um funcionário, descontando 10% de imposto.
- **Fórmula:** Salário Líquido = Salário Bruto - (Salário Bruto * 10/100)

### 10. Conversor de Minutos para Horas
Crie um programa que converta minutos em horas e minutos.
- **Fórmula:** Horas = minutos / 60, Minutos Restantes = minutos % 60

### 11. Calculadora de Desconto Simples
Escreva um programa que aplique um desconto de 10% em um produto.
- **Fórmula:** Valor com Desconto = Valor Original - (Valor Original * 10/100)

### 12. Verificador de Número Positivo ou Negativo
Crie um programa que verifique se um número é positivo ou negativo.

### 13. Calculadora de IMC Simplificada
Escreva um programa que calcule o IMC de uma pessoa e classifique-a como "Abaixo do peso", "Peso normal", ou "Sobrepeso".
- **Fórmula:** IMC = peso (kg) / altura² (m)

### 14. Contador de Caracteres
Escreva um programa que conte quantos caracteres existem em uma string.

### 15. Simulador de Caixa Eletrônico
Crie um programa que simule um caixa eletrônico, permitindo saques e depósitos.
